Curriculum

Two Real Health AI Problems

Both problems use real datasets. Both demand real engineering. Both teach you how researchers think.

🧬
Problem 1 — DNA Sequence Classification
Kaggle · Multi-species Gene Family Dataset
  • 1.1 EDA — Load & Explore DNA Data
  • 1.2 Feature Engineering — k-mer Frequency Extraction
  • 1.3 Dimensionality Reduction — PCA & UMAP
  • 1.4 Feature Selection — LASSO (L1 Regularization)
  • 1.5 Tabular ML — XGBoost (Main Model)
  • 1.6 Model Comparison — LightGBM · CatBoost · Random Forest
  • 1.7 Interpretability — SHAP Values
  • 1.8 Scoreboard & Evaluation Summary
🫀
Problem 2 — Heart Attack Risk Prediction
UCI · Clinical Cardiovascular Dataset
  • 2.1 EDA — Load & Explore Heart Attack Data
  • 2.2 Class Imbalance — Class Weights & SMOTE
  • 2.3 Feature Selection — LASSO & Elastic Net
  • 2.4 Tabular ML — XGBoost Tuning & Early Stopping
  • 2.5 Model Comparison Scoreboard
  • 2.6 Interpretability — SHAP + Permutation Importance
  • 2.7 Survival Analysis — Cox Proportional Hazards
  • 2.8 Final Scoreboard & Clinical Insights
Clinical Disclaimer: These are educational models built for learning. They are not intended for real clinical decision-making and require proper clinical validation before any medical use.

Technology

Full Stack of Tools & Libraries

Industry-standard tools used by data scientists in health research and clinical AI.

Tool / Library Role in Workshop Key Concepts
XGBoost Main classification model for both problems Gradient boosting, early stopping, class weights
LightGBM Model comparison baseline Leaf-wise tree growth, fast training
CatBoost Model comparison baseline Categorical feature handling, symmetric trees
SHAP Explainability for both problems Global + local feature importance, beeswarm plots
scikit-learn Core ML pipeline and evaluation PCA, LASSO, cross-validation, metrics
imbalanced-learn Handle class imbalance in heart data SMOTE oversampling, resampling strategies
lifelines Survival analysis — Problem 2 Cox PH model, Kaplan-Meier, log-rank test
Biopython DNA sequence parsing Nucleotide sequences, k-mer extraction
UMAP Dimensionality reduction visualization Non-linear manifold learning
Plotly · Seaborn · Matplotlib All data visualizations Interactive charts, confusion matrices, ROC curves
